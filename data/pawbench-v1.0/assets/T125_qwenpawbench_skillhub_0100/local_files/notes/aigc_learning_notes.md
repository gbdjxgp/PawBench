# AIGC 视频创作学习笔记

## 2024-03-15 学习记录

### 儿童科普视频要点
- 时长控制在 1-2 分钟（小朋友注意力有限）
- 语言要简单、重复关键概念
- 画面要鲜艳、动作要夸张
- 加入互动问题："你看到了什么颜色？"

### 分镜设计原则
1. 开场 3 秒必须吸引注意力
2. 每个场景不超过 5-8 秒
3. 旁白要配合画面节奏
4. 结尾要有小总结

### 我需要用到的工具
- video-script-generator：生成脚本和分镜
- 后续可能用：digital-avatar（口播）、scene-video-generator（画面）
